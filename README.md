# Cryptocurrency Market Analysis Dashboard | Power BI

## Project Overview

This project presents an interactive Power BI dashboard developed using a Cryptocurrency Market Dataset sourced from Kaggle.

The dashboard analyzes cryptocurrency performance, market trends, trading activity, and price movements to provide actionable insights through data visualization and business intelligence techniques.

---

## Dataset

Source: Kaggle Cryptocurrency Dataset

The dataset includes cryptocurrency market information such as:

* Cryptocurrency Name
* Symbol
* Market Capitalization
* Open Price
* Close Price
* High Price
* Low Price
* Trading Volume
* Market Trends
* Historical Price Data

---

## Business Objectives

The dashboard was designed to answer key analytical questions:

* Which cryptocurrencies have the highest market capitalization?
* Which coins show the highest trading volume?
* How have cryptocurrency prices changed over time?
* What are the top-performing cryptocurrencies?
* Which cryptocurrencies experienced the highest gains and losses?
* What trends can be observed in the crypto market?
